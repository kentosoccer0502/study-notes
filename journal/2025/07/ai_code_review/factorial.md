## 以下に再帰を使用した実装例を示します。

```typescript
function factorial(n: number): number {
  if (n <= 1) {
    return 1;
  }
  return n * factorial(n - 1);
}
```

## 再帰を使った実装のポイント

- ベースケース: n が 1 以下の場合、1 を返すことで再帰を終了します。

- 再帰ステップ: n を n-1 の階乗に掛けることで、問題を小さくしていきます。

この再帰的なアプローチは、問題の指示に従っており、数学的な階乗の定義に基づいています。
